Welcome to Beat 1.4 beta!

What's New?

  • Lots of optimization and bug fixes
  • Typewriter Mode
  • New section & synopsis layout

Typewriter Mode

Keeps the caret and selection at the center of the screen. You can access it by toggling View → Typewriter Mode on.

New Section Layout

Sections and synopsis markers now use the system font to make them stand out from the actual script.
